Open in mobile SoundCloud App

Directly open a SoundCloud track in the mobile app,
via redirection to a special URL.
Should work on iOS and Android.

Use with URL

Just change https://soundcloud.com to http://soundcloudapp.ars.is
(replace .com with app.ars.is
and https with http).

Use with Bookmarklet

Uses heroku domain, so HTTPS works!
(https://soundcloud-open-in-app.herokuapp.com)

  • Drag this to your Bookmarks bar:

    OpenInSoundCloudApp

  • Or copy this text and save it as a bookmark:

    WTF? Read "Bookmarklet" on Wikipedia

API How does it work?

{slug}

Parameter: just the "slug" of a track's URL (i.e. the part after soundcloud.com).

Redirects to a special URL (soundcloud://tracks:{id}),
which will open the SoundCloud App on iOS and Android devices.

Can be used without a bookmarklet, by just editing the URL in your browser.
Just change soundcloud.com to soundcloudapp.ars.is (replace .com with app.ars.is).

Important: HTTP only!

URL:

http://soundcloudapp.ars.is/go/{soundcloud_url}

Example Link

http://soundcloudapp.ars.is/ninja-tune/solid-steel-radio-show-20-12-1
